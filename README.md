# OrangePI 6 Plus NPU Inference Setup and Testing

Setup and test script for those who want to setup and test CiX NPU on OrangePI 6 plus boards.

This utility will perform the following steps:

1. Install necessary system dependencies
2. Set up Python environment using pyenv
3. Clone the [CIX AI Model Hub repository](https://www.modelscope.cn/cix/ai_model_hub)
4. Install Python dependencies for the Model Hub
5. Install the Zhouyi NPU Runtime and NOE Engine

# Requirements

It is recommended using the official Ubuntu 24.04 OrangePI 6 plus image available at http://www.orangepi.org/html/hardWare/computerAndMicrocontrollers/service-and-support/Orange-Pi-6-Plus.html.

# How to use it

After clone this repository, execute `setup_and_test.sh`.

This project is a fork of https://github.com/mrobinson-opi/6plus_npu.
